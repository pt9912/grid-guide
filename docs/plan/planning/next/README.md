# Geplante Arbeit (Scope skizziert)

Eintraege, deren Scope skizziert ist, die aber noch nicht aktiv
bearbeitet werden.

Lebenszyklus und Verzeichnisstruktur sind in
[`ADR 0001`](../../adr/0001-documentation-and-planning-structure.md)
§2.1 definiert.

## Konvention fuer Eintraege

- Dateiname `NNN-kurz-titel.md` mit dreistelliger Nummer (ggf. aus
  `open/` uebernommen).
- Jeder Eintrag enthaelt Ziel, grobes Scope, Akzeptanzkriterien und
  Vorbedingungen fuer die Aktivierung.
