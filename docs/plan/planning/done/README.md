# Abgeschlossene Arbeit

Archiv der gelieferten Slice-Plaene und Roadmap-Phasen mit kurzer
Closure-Notiz (was wurde geliefert, was bleibt offen).

Lebenszyklus und Verzeichnisstruktur sind in
[`ADR 0001`](../../adr/0001-documentation-and-planning-structure.md)
§2.1 definiert.

## Konvention fuer Eintraege

- Dateiname behaelt die Slice-Nummer aus `in-progress/`.
- Closure-Notiz steht am Ende des Dokuments unter `## Closure`.
- Wird ein gelieferter Plan spaeter komplett ueberholt, wandert er
  nach `docs/archive/`.
